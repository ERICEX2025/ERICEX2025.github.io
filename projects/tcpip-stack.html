Virtual TCP/IP Network Stack

Sep 2024 – Nov 2024

Built a virtual network stack in Go from scratch, supporting IP forwarding, RIP-based routing, dynamic routing table updates, and an RFC-compliant TCP protocol with state machine, sliding window, retransmission, and congestion control.

Network Layer

  • ● IP Forwarding: Implemented Internet Protocol packet forwarding with proper header parsing, checksum validation, and TTL (Time To Live) handling
  • ● RIP-based Routing: Built a dynamic routing protocol using the Routing Information Protocol (RIP), enabling routers to discover and maintain optimal paths through the network
  • ● Dynamic Routing Tables: Implemented automatic routing table updates that adapt to network topology changes, ensuring efficient packet delivery across the network

Transport Layer

  • ● TCP State Machine: Implemented the complete TCP connection lifecycle, including connection establishment (three-way handshake), data transfer, and connection termination
  • ● Sliding Window Protocol: Built flow control mechanisms using sliding windows to manage data transmission and ensure reliable delivery
  • ● Retransmission: Implemented automatic packet retransmission with exponential backoff to handle network packet loss
  • ● Congestion Control: Developed congestion control algorithms (including slow start and congestion avoidance) to adapt transmission rates based on network conditions
  • ● Socket API & Testing REPL: Developed a socket API and testing REPL to establish, manage, and verify reliable data transfer across lossy virtual networks
